信息工作台
功能 API 技术栈

高效阅读，智能管理

基于 Django + Tailwind CSS 构建的信息工作台，支持 RSS 订阅、API 新闻导入、AI 写作笔记和外部推送接口。

开始使用 API 文档

核心功能

RSS 订阅

添加和管理 RSS 订阅源，支持 36kr、少数派等主流站点，自动获取最新文章。

新闻资讯

支持 API 导入新闻，兼容 NewsAPI、GNews 等，多来源管理和分类筛选。

笔记系统

支持 Markdown 格式，标签管理，置顶和归档，与文章关联。

AI 写作

集成 OpenAI GPT，支持文章总结、改写、扩写和多语言翻译。

外部 API

RESTful API 接口，支持外部应用推送新闻，API 密钥认证。

仪表盘

直观的统计信息，最新文章和笔记一览，快速访问常用功能。

API 接口

推送新闻

允许外部应用通过 API 推送新闻到工作台数据库

POST /api/v1/news/push/
Headers:
  X-API-Key: your-api-key
  Content-Type: application/json

Body:
{
  "title": "新闻标题",
  "link": "https://example.com/article/123",
  "content": "新闻内容...",
  "author": "作者名",
  "category": "科技"
}

批量推送

POST /api/v1/news/push/batch/
Headers:
  X-API-Key: your-api-key
  Content-Type: application/json

Body:
{
  "articles": [
    {"title": "标题1", "link": "链接1"},
    {"title": "标题2", "link": "链接2"}
  ]
}
